# Mindful Coffee - 带睡眠预测和皮质醇节律功能的iPhone咖啡因追踪App

**追踪咖啡因、预测睡眠、全天优化精力的最智能方式。**

不再猜测何时能入睡。Mindful Coffee是一款设计精美的iOS应用，追踪你的咖啡、茶和能量饮料摄入，同时精确告诉你咖啡因何时会从体内清除，让你享受高质量睡眠。理解身体的自然皮质醇节律，完美把握咖啡因摄入时机，达到最佳状态。

采用Apple绚丽的**Liquid Glass**设计语言，Mindful Coffee在你的iPhone上呈现现代半透明界面和丝滑动画，完美融入iOS生态。

## 为什么选择Mindful Coffee?

大多数咖啡因追踪器只是简单计算毫克数。Mindful Coffee更深入一步：它模拟你的个人代谢，可视化皮质醇觉醒反应，并基于真实科学预测你的最佳睡眠时间。无论你是想优化晨间routine的咖啡爱好者，还是希望改善睡眠质量的人，这款应用都能给你所需的洞察。

## 核心功能

**智能记录，安然入睡**

- **每日记录连续天数** - 可视化计数器庆祝你的坚持，保持动力
- **睡眠洞察卡片** - 一目了然查看昨晚睡眠时长和咖啡因影响
- **分组历史记录** - 当月展开显示，往月紧凑汇总，轻松浏览

**智能咖啡因追踪**

- **一键记录** - 瞬间记录咖啡、茶、能量饮料和自定义饮品
- **智能排序** - 按"最近使用"或"最常使用"排序，记录更快捷
- **实时睡眠预测** - 精确知道何时可以享受优质睡眠
- **假设模拟器** - 在喝之前预览再来一杯会如何影响入睡时间

**为你的身体量身定制**

- **个人代谢测验** - 回答引导式问题，校准敏感度、半衰期和睡眠阈值
- **皮质醇节律建模** - 可视化身体的自然能量曲线，优化咖啡因时机
- **睡眠相关性分析** - 利用Apple健康数据，发现咖啡因水平如何影响实际睡眠时长

**与Apple无缝集成**

- **HealthKit同步** - 自动将咖啡因记录到Apple健康，导入睡眠数据获取更深洞察
- **从其他App导入** - 通过HealthKit导入第三方应用的咖啡因数据
- **CSV导出** - 完整历史导出，数据完全由你掌控

## 隐私至上

你的数据留在你的设备上。Mindful Coffee使用SwiftData将所有消费数据本地存储。HealthKit访问需要你的明确授权，遵循Apple准则。**所有计算、分析和建模完全在设备上运行** - 绝不发送到任何服务器。
