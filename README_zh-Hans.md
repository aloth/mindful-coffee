# Mindful Coffee：咖啡因日志 & 皮质醇节律

**追踪咖啡因摄入，可视化皮质醇节律，优化精力与睡眠。**

Mindful Coffee 是一款智能 iOS 应用，帮助你监测每日咖啡因摄入，并深入了解它与身体自然能量曲线之间的联系。记录咖啡、茶和其他含咖啡因饮品，获取关于**皮质醇节律**、睡眠模式和整体健康的宝贵洞察。做出更明智的选择——享受更安稳的夜晚和更高效、专注的白天。

Mindful Coffee 采用 Apple **Liquid Glass** 设计美学，在最新 iPhone 上呈现惊艳效果：现代的半透明界面元素和流畅动画，完美融入 iOS 生态。

## ✨ 核心功能

* **全新！每日记录连续天数：** 可视化的连续天数计数器，让你始终保持记录动力，庆祝每一天的坚持。
* **全新！睡眠洞察卡片：** 一目了然地查看昨晚睡眠时长和当天咖啡因摄入，并与你的平均睡眠时长对比。
* **全新！分组历史记录：** 轻松浏览消费历史——当月记录完整展开，往月数据整洁汇总，快速查阅。
* **轻松且详细的记录：** 通过友好的界面快速记录咖啡因摄入。完成记录后自动返回"今日"屏幕，即时掌握整体情况。自定义你的饮品列表：
    * **智能排序：** 按"最近使用"或"最常使用"排序，记录更快捷。
* **皮质醇节律建模：** 查看身体的自然能量曲线（皮质醇觉醒反应 CAR），优化咖啡因摄入时机，提升专注力和健康感。了解每杯饮品如何影响这一关键节律。
* **智能睡眠预测：** 根据你的摄入量和个性化设置，实时预估咖啡因何时降到足以保证优质睡眠的水平。
* **"假设"模拟：** 在饮品选择列表中，即时预览再喝一杯会如何影响预计入睡时间——*在你喝之前*就能看到。
* **个人代谢助手：** 回答一系列关于咖啡因体验的引导式问题，助手会分析你的答案，推荐个性化的敏感度、半衰期和阈值设置，让 App 更精准地匹配你的身体。
* **个性化代谢与皮质醇模型：** 调整半衰期、敏感度、睡眠阈值，以及皮质醇曲线的核心参数，让计算精准贴合你的身体。
* **睡眠相关性分析：** 分析不同每日咖啡因摄入量与睡眠时长的关联（睡眠数据从 Apple 健康导入），并可自定义咖啡因阈值以获得更有意义的个人洞察。
* **智能每日记录提醒（可选）：** 仅在当天尚未记录时，发送温和提醒。
* **HealthKit 集成：**
    * **写入：** 自动将咖啡因摄入和饮品体积（作为水分）写入 Apple 健康。
    * **读取：** 读取睡眠分析数据，用于更深入的关联分析。
    * **可选导入：** 从 Apple 健康导入之前由第三方 App 记录的咖啡因数据。
* **数据管理：** 通过 CSV 文件**导入**和**导出**完整的摄入历史。
* **高度可定制：** 设置偏好的就寝时间，选择喜欢的主题（跟随系统、浅色、深色）。
